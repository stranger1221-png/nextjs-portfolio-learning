# Next.js Portfolio Learning

This is a learning project focused on the fundamentals of Next.js. The aim is to develop a portfolio application while covering the most important concepts and features of Next.js and React in a practical, hands-on way.

## Goals
- Understand the basics of Next.js, including App Router, routing, server components, and client components.
- Build a portfolio to showcase personal projects and skills
- Learn how React works inside Next.js: hooks, components, props, state, and more
- All code is heavily commented with explanations for every file, component, and concept

## Learning Outcomes
- Practical, production-level Next.js app structure
- Custom components, hooks, data fetching, forms, routing, and more
- Best practices for modern Next.js/React development
